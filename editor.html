Editor Konten

Generate HTML siap-paste untuk update website tanpa coding

Cara Pakai

  1. Pilih tab konten yang ingin ditambahkan
  2. Isi form — lalu klik Generate HTML
  3. Klik Salin Kode untuk menyalin hasilnya
  4. Buka file yang dituju di GitHub → klik ikon pensil ✏️ → paste kode → Commit

Tambah Berita Baru

Generate kartu berita untuk ditempel di news.html — cari komentar <!-- NEWS CARDS --> atau sebelum div add-news-box

Paste ke: news.html — sebelum <!-- ADD NEWS -->

Tambah Publikasi Ilmiah

Generate kartu publikasi untuk ditempel di publications.html — cari komentar <!-- Publication List -->

Paste ke: publications.html — di dalam <div id="pub-list">

Tambah Proyek Penelitian

Generate kartu proyek untuk ditempel di research.html — cari komentar <!-- Research Projects -->

Paste ke: research.html

Tambah Kegiatan Pengabdian

Generate kartu kegiatan untuk ditempel di pengabdian.html — cari komentar <!-- Activity Cards -->

Paste ke: pengabdian.html — di dalam grid aktivitas

Tambah Produk / Prototipe

Generate kartu produk untuk ditempel di products.html — cari <div id="products-grid">

Paste ke: products.html — di dalam <div id="products-grid">

Tambah Anggota Tim

Generate kartu anggota untuk ditempel di team.html — cari bagian divisi yang sesuai

Paste ke: team.html — di dalam grid divisi yang sesuai
Tip Foto: Simpan foto di assets/images/ dengan nama sesuai yang diisi di atas (contoh: nama-anggota.jpg). Ukuran ideal 300×300 px. Jika tidak ada foto, inisial nama akan tampil otomatis.
